Getting Started

Burme AI is a multi-provider AI API platform with automatic failover. To get started:

  • Log in with your admin or user credentials
  • Navigate to the Chat page to start chatting
  • Use the Dashboard to manage users
  • Check API Status to monitor provider health

API Endpoints

POST /api/chat
message string The chat message
history array Previous messages (optional)
model string Model to use (optional)
GET /api/users
POST /api/users
username string New username
password string New password

API Providers

Burme AI uses the following providers in failover order:

  • Groq - High-speed inference
  • Cerebras - Ultra-fast compute
  • OpenRouter - Aggregated access
  • NVIDIA - Enterprise-grade
  • HuggingFace - Open source models

Environment Variables

Configure API keys in your Vercel project settings:

GROQ_API_KEY=your_groq_key
CEREBRAS_API_KEY=your_cerebras_key
OPENROUTER_API_KEY=your_openrouter_key
NVIDIA_API_KEY=your_nvidia_key
HUGGINGFACE_API_KEY=your_huggingface_key
SECRET_KEY=your_secure_secret_key

For multiple keys, separate with commas:

GROQ_API_KEY=key1,key2,key3
